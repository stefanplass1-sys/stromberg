### Stromberg-Sprüche-App ###

Erzeuge eine Web Applikation auf Basis von Java / Vaadin / Spring Boot / Maven zur Darstellung eines Spruches des Tages. Verwende folgende Zutaten:

- das Layout von Claude Design im Verzeichnis "stromberg"
- die Sprüche im Verzeichnis "Sprüche"
- zu jedem Spruch ein Foto aus dem Verzeichnis "Fotos"
