## Hintergrund

Die Comedy-Fernsehserie „Stromberg" lief von 2004 bis 2012 über fünf Staffeln auf ProSieben. Im Stil einer Doku-Soap begleitet die Serie den Büroalltag in der Abteilung Schadensregulierung der fiktiven „Capitol Versicherung AG". Das Konzept ist an die britische Serie „The Office" angelehnt.
